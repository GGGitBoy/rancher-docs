## Overview

Rancher Prime integrates with the Azure Marketplace as a pay-as-you-go (PAYG) offering. This brings the value of running and managing Kubernetes environments to Azure customers, with the benefit of a new pay monthly pricing model available through the Azure Marketplace. This listing will enable you to manage any CNCF-certified Kubernetes distribution in Azure, on-prem, or at the edge. To learn more, see our [Azure Marketplace offering](https://azuremarketplace.microsoft.com/en-us/marketplace/apps/suse.rancher-prime-llc?tab=Overview).
